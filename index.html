Typography rules
  • Always sentence case for headings — never title case or all caps (eyebrow labels excepted)
  • Maximum 3 font weights on any single page or spread
  • Do not italicize headings as an emphasis style
  • Create hierarchy through weight variation, not size alone

Brand rules

Logo
Do

Use approved Box files. Maintain clear space on all sides. Match the logo variant to the background.

Don't

Recreate or approximate the logo. Add drop shadows, glows, or effects. Stretch, skew, rotate, or recolor. Place below the 85px minimum.

Color
Do

Lead with Coral for all CTAs. Use Squid for dark backgrounds. Use the Wine-to-Coral gradient for decorative accents. Limit to 3 colors per layout.

Don't

Use Mist as a primary page background. Set Coral text on Wine. Introduce colors outside the approved palette.

Typography
Do

Use Suisse Int'l for all branded communications. Use sentence case for all headings. Build hierarchy through weight.

Don't

Use more than 3 weights on a single page. Use title case or all-caps for headings. Italicize headings.

Photography & illustration

RDC photography should feel candid, human, and professional — never stock-photo staged. All assets are maintained in Box under Red Door File Database → Red Door Guidelines and Assets → RDC 2025 Branding.

You are creating an HTML deliverable for Red Door Collaborative (RDC). Apply the RDC design system precisely. Do not approximate colors, invent new components, or use fonts outside this spec. How to use
  1. Open a new Claude conversation
  2. Paste the system prompt above as your first message (or add to your Claude project instructions)
  3. Describe the HTML page you need — Claude will apply all brand rules automatically
  4. The generated HTML links to rdc.css from the live GitHub Pages URL, so it renders correctly in any browser
Tip: Save the system prompt as a Claude Project instruction so it applies automatically to every conversation in that project.
